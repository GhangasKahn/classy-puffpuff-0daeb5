# BEDROCK Visual System v2 — The Unreliable Ledger

Full-stack frontend redesign: landing + PWA share one industrial language.

## DNA

| Source | What we took |
|---|---|
| **Brutalist** | Zero radius, 2px ink borders, oversized display type, exposed structure |
| **Grunge** | Film grain, diagonal hatch fields, imperfect atmosphere |
| **Nothing** | Glyph diamond markers, mono telemetry, transparent glass chrome |
| **Teenage Engineering** | Modular panels, engineering labels (`01 / DOCTRINE`), TE orange `#FF4D00`, hit-shadow buttons |
| **Virgil / Off-White** | Quotation marks on key words, `™`, caution-tape stripe `#E8FF00`, industrial Helvetica energy via Archivo Black |

## Tokens

Shared file: `site/brand/bedrock-system.css`

- **Default theme:** light industrial (`#F7F7F5`) — not cream, not purple
- **Dark theme:** Nothing black (`#0A0A0A`)
- **Type:** Archivo Black (display) · Chakra Petch (UI eng) · Archivo (body) · Space Mono (data)
- **Motion:** hard offset hover (`translate -2px` + ink shadow), grain, GSAP mask reveals
